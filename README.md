# 560-ViT-Finetuned
COMP560 Artificial Intelligence Final Project: Vision Transformer to Identify AI-Generated Faces

##Purpose
This project fine-tunes a Vision Transformer (ViT) to classify the difference between real human faces and ai-generated faces generated by models like Stable Diffusion and StyleGAN. 

## Tools
- Model: microsoft/beit-base-path16-224 (BEiT)
- Frameworks and libraries: PyTorch, Hugging Face ```transformer```
