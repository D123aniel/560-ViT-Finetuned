COMP560 Artificial Intelligence Final Project: Vision Transformer to Identify AI-Generated Faces

Authors: Reed Lai, Aidan Yung, Daniel Zhang

Spring 2026

Presentation: https://docs.google.com/presentation/d/16QBHkJDWMtrFbS033Nngcx_agF6flb_wT1jCOu05YWk/edit?usp=sharing

## Purpose
This project fine-tunes a Vision Transformer (ViT) to classify the difference between real human faces and ai-generated faces generated by models like Stable Diffusion and StyleGAN. 

[Link to Model:](https://huggingface.co/D123aniel/560_ViT_Finetuned)
[Link to Dataset:](https://huggingface.co/datasets/D123aniel/560_ViT_dataset)

## Tools
- Model: microsoft/beit-base-path16-224 (BEiT)
- Frameworks and libraries: PyTorch, Hugging Face ```transformers```, ```datasets```, ```evaluate```
- Hardware: NVIDIA RTX 5070TI (16GB VRAM), Ryzen 5 9600X (6-core 12-thread), 32GB DDR5 RAM

## Process
1. Sourced 10k images, 5k real faces, 5k ai-generated faces. Dataset split into 60% train, 20% validation, 20% test.
2. Fine-tuning: Supervised fine-tuning on the BEiT model using Huggin Face ```Trainer``` API, using train and validation datasets.
3. Image Processing: Used ```AutoImageProcessor``` for image standardization and feature extraction.
4. Evaluation: Use test dataset to validate the model on unseen faces.

## Files
- ```prep_dataset.ipynb```: Shuffle data within real/fake, then split into train-validate-test, and upload complete dataset to Hugging Face
- ```train-model-lite.ipynb```: Used to test environment and setup on a smaller dataset
- ```train-model-splice-data.py```: Main file used to fine-tune the BEiT model on the whole dataset
- ```test_accuracy.py```: Script to test fine-tuned model on the test dataset
- ```test_image.ipynb```: Add own image link to classify as real/fake with fine-tuned model
- ```test_nvidia.py```: Validate that system has correct dependencies installed and can use NVIDIA GPU for training
- ```upload-model.py```: Upload fine-tuned model to Hugging Face

## Usage Notes
- Ensure HF_TOKEN environment variable is set
- This code is written for a system utilizing Windows 11 and an NVIDIA RTX 5070TI, or any NVIDIA Blackwell GPU
